# 用户指南概述

欢迎来到 Agentrix 用户指南。本文档将帮助您开始使用 Agentrix 平台并充分利用其功能。

## 什么是 Agentrix？

Agentrix 是一个企业级 AI 代理托管和协作平台，使团队能够：

*   创建 AI 任务：将工作分配给专业的 AI 代理
*   本地执行：使用 CLI 在您的本地计算机上安全运行任务
*   Git 集成：与 GitHub 和 GitLab 无缝集成
*   协作：共享结果并与团队成员合作
*   企业安全：端到端加密和安全数据处理

## 入门指南

### 先决条件

在使用 Agentrix 之前，您需要：

*   一个现代操作系统（macOS、Linux 或 Windows with WSL）
*   系统上已安装 Git
*   Node.js 18+（用于 CLI）
*   一个 Agentrix 账户

## 核心功能

### 任务管理

学习如何创建、管理和组织您的 AI 任务：

*   创建具有详细要求的任务
*   将任务分配给特定代理
*   设置优先级和截止日期
*   添加上下文文件和存储库
*   跟踪任务进度

### 本地执行器 CLI

Agentrix CLI 允许代理在您的本地计算机上安全运行：

*   从平台拉取任务
*   以完全访问本地环境的方式执行
*   提交前审查变更
*   将结果推回平台

### Git 集成

与您现有的 Git 工作流程无缝集成：

*   连接 GitHub 或 GitLab 仓库
*   创建自动拉取请求
*   在您喜欢的工具中审查更改
*   保持完整的 Git 历史记录

### 协作

与您的团队一起工作：

*   与团队成员分享任务结果
*   为外部观众创建可分享链接
*   评论和讨论任务输出
*   跟踪团队生产力

### 安全与隐私

企业级安全功能：

*   敏感数据的端到端加密
*   本地执行以确保最高安全
*   细粒度访问控制
*   审计日志和合规性

## 常见任务

### 创建任务

```bash
# From the web interface or CLI
agentrix task create \
  --title "Fix authentication bug" \
  --description "Debug and fix the login issue" \
  --repo "github.com/myorg/myapp" \
  --priority high
```

### 本地运行任务

```bash
# Pull and execute pending tasks
agentrix pull
agentrix exec

# Review and commit changes
agentrix status
agentrix commit
```

### 分享结果

```bash
# Create a shareable link for task results
agentrix share create --task-id task_123 \
  --password secure123 \
  --expires-in 7d
```

如果您遇到问题：

    *   常见安装问题

    ```bash
    agentrix doctor
    ```
    
3.  **查看日志**
    
    ```bash
    agentrix logs --tail 50
    ```
    
4.  **重置配置**
    
    ```bash
    agentrix config reset
    ```
