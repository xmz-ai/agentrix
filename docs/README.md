# Agentrix Documentation

> Enterprise AI Agent Hosting and Collaboration Platform

## Documentation Structure

This documentation is organized for three distinct audiences:

### [User Guide](./en/user-guide/)
For Agentrix platform users, product managers, and business experts.
- Getting started with Agentrix
- Creating and managing tasks
- Using local executor (CLI)
- Git integration with GitHub/GitLab
- Security and encryption

### [Agent Developers](./en/agent-developers/)
For developers building custom agents.
- Agent structure and configuration
- Hook system (10 hook types)
- MCP servers and skills integration
- Testing and best practices
- Complete examples

### [Platform Developers](./en/platform-developers/)
For Agentrix core development team.
- System architecture
- API/CLI/App development
- Deployment and operations
- Development workflows
- Contributing guidelines
